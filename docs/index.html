Rainbow V Manager

A Textual-based TUI (Terminal User Interface) application for managing QEMU/KVM virtual machines using the libvirt python API.

Key Features

  • Manage VMs via terminal without X11
  • Remote control with low bandwidth
  • Features like disabling disks and changing machine types
  • Supports multiple libvirt server connections
